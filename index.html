Radiomics Analysis Portal sitkCore¶

This sitkCore for the Radiomics Analysis Portal (RAP) contains common utilities for developing algorithms for analysis of data from Tuberculosis Portals (TBPortals). The tools are written in Python and provided as an installable Python package. The application programming interface (API) is documented in Sphinx: https://sturdy-winner-164e7c64.pages.github.io/
